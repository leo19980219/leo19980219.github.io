Exercise Recorder

This is final project of Signal and System.

Introduction

This project combined Matlab with Matlab mobile which has powerful and convenient sensor system. In this app, I took advantage of "Position" sensor. With Matlab mobile, it is extremely easy to store the sensor data and access from the Matlab Driver if using MathWorks Cloud as develoment envrinment.

In this app, you can see your exercise path in a map as well as average speed, movement distance and exercise duration. I record my own exercise for example, walking from NTPU libraby to youbike station ,and riding the bicycle in the campus. They are all clearly showed in demo video and the below picture.

Tools & Envrinment

Collect Data Using Matlab Mobile IOS
Develope Using MathWorks Cloud
Store Data in Matlab Drive

Demo

Step1: Collect data by Matlab Mobile.
Open Matlab Mobile and switch to "sensor" page.
Select "Stream to log" and "Sample rate 10.0Hz".
Remeber to enable "Position".
Press "START" to create your own record, you can also name your record whatever you like.

Step2: Open Matlab APP.
In APP, you can type which record you want to see.
Press "Start" to see the result.
